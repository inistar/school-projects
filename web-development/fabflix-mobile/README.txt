## Web Development project from CS122B Class. (Android App)
Developed a real, Web-based commercial web site to sell movies using techniques such as database connectivity, 
web application development, and database administration. Website allows customer to browse/search for movie information
and buy movies on both desktop and Android devices. 

HTML/Bootstrap, CSS (Frontend)
Java Servlets(JSP), MySQL (Backend)
Java (Android Studio)

Hosted on AWS EC2 using Apache Tomcat.
